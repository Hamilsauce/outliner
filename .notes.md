# OUTLINER
## Model
### Data Structures
- A primary infinitely-nested list allowing
 zooming/focusing in on any nodes

- Secondary user defined lists composed of
  nodes moved or copied from main list

- ROOT Node - list of lists

### Possible Operations
#### Creation: 
- Insert
- Add
- Attach
- Create

#### Modify: 
- Update
- Edit

#### Copy: 
- Update
- Edit

#### Delete: 
- remove
- delete
- undo

#### Move: 
- Promote
- Demote
